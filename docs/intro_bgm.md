# 背景音乐控制

可以播放背景音乐，会自动无限循环，同时只能有一个背景音乐会播放
注意，链接开头的两个 // 都需要加上 \

```
例如
https://www.ahayoo.com/abc
需要写成
https:\/\/www.ahayoo.com/abc
```

使用方法：
```
播放
# BGM: <音乐远程路径或本地相对路径>
例如
# BGM: http:\/\/downsc.chinaz.net/Files/DownLoad/sound1/201906/11582.mp3
# BGM: my_music.mp3

停止 
# BGM: stop

暂停 
# BGM: pause

恢复 
# BGM: resume
```
